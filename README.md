# algeria-tourism-platform
Tourism web platform showcasing Algerian regions with admin panel (PHP, MySQL, JS)
# Frontend
HTML5 → structure des pages
CSS3 → design (layout, responsive, animations)
JavaScript → interactions
jQuery → animations 

# Backend
PHP → logique serveur
gestion des sessions ($_SESSION)
pages dynamiques (.php)

# Base de données
MySQL → stockage des données
(utilisateurs, admin, etc.)
